# 🗺️ Roadmap du Projet

Ce document suit l'évolution du projet Green Mood CBD, du prototype à la version de production.

## ✅ Phase 1 : Fondations & E-commerce (Terminé)
- [x] Configuration de l'environnement React 19 + Vite + Tailwind 4.
- [x] Structure de la base de données Supabase (Produits, Catégories, Profils).
- [x] Système d'authentification utilisateur.
- [x] Catalogue produit avec filtres et recherche.
- [x] Panier et gestion des stocks.

## 🚧 Phase 2 : BudTender IA & Voice (En cours)
- [x] Intégration de Gemini Live API (Text & Voice).
- [x] Création du widget conversationnel premium.
- [x] Tool calling : permettre à l'IA d'ajouter des produits au panier.
- [/] Rafinement des prompts de l'IA pour expertise CBD spécialisée.
- [/] Amélioration de la fluidité vocale (gestion des interruptions).

## 📈 Phase 3 : Fidélité & Récurrence (Prochainement)
- [ ] **Système de Parrainage** : Génération de codes et suivi des transformations.
- [ ] **Abonnements** : Mise en place de commandes automatiques mensuelles.
- [ ] **Avis & Reviews** : Permettre aux clients de noter les produits achetés.
- [ ] **Dashboard Admin V2** : Statistiques de vente avancées et gestion des flux de stock.

## 🚀 Phase 4 : Expansion & Marketing
- [ ] Intégration de paiement réelle (Viva Wallet / Stripe).
- [ ] Notifications Push pour le suivi des commandes.
- [ ] Optimisation SEO avancée pour chaque produit.
- [ ] Version mobile native (PWA ou React Native).
